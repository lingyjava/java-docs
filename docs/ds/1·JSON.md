## JSON是什么
JSON是一种特殊的字符串，是不同种类客户端和服务器端交互通用格式，使程序能在移动端和电脑端使用。

## 格式
JSON字符串是键和值成对出现的，键是字符串，严格的json格式键的两端必须加双引号。
```json
//json对象{}
//使用/对双引号进行转义
String json = "{\"id\":12,\"name\":\"张三\",\"pwd\":\"123456\"}";

//json数组[]
String[] json = 
"[
	{\"id\":12,\"name\":\"张三\",\"pwd\":\"123456\"},
    {\"id\":12,\"name\":\"李四\",\"pwd\":\"123456\"},
    {\"id\":12,\"name\":\"王五\",\"pwd\":\"123456\"}
]";
```
