## 终端命令
查看终端执行过的命令
history

查看端口占用情况
lsof -i:[3000]

杀死进程
kill [pid]

截图工具
comment + shift + 5 

显示隐藏文件
comment + shift + . 

打开一个新的终端窗口。IDEA中的生成代码。
comment + n 

打开强制退出应用窗口
comment + alt + esc

查看磁盘空间
df -h

查看inode空间
df -i
